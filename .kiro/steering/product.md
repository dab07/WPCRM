# Product Overview

## WhatsApp CRM System

An AI-powered WhatsApp CRM system designed for customer relationship management with automated responses and agent handover capabilities.

### Core Features

- **AI Auto-Reply**: Intelligent intent detection with automated responses for common customer inquiries (greetings, thanks, business hours)
- **Real-time Chat Interface**: 1:1 conversations with customers through WhatsApp integration
- **Contact Management**: Comprehensive contact database with tagging, search, and filtering
- **Campaign Management**: Bulk messaging for marketing campaigns with targeting and scheduling
- **Follow-up Automation**: Rule-based automated follow-ups based on conversation inactivity
- **Business Card Processing**: OCR integration for automatic contact creation from business card images
- **Agent Assignment**: Multi-agent support with conversation assignment and handover

### Access Model

- **No Authentication**: Direct access to all agentic AI features
- **Single System**: All users have full access to manage conversations, contacts, campaigns, triggers, and workflows

### Key Workflows

1. **Incoming Messages**: WhatsApp → AI Processing → Auto-reply or Trigger Activation
2. **Trigger-Based Automation**: Event detection triggers n8n workflows automatically
3. **Contact Creation**: Manual entry or automatic from conversations
4. **Campaign Execution**: Bulk messaging with targeting and delivery tracking
5. **Follow-up Processing**: Automated scheduling based on conversation inactivity rules
6. **Agentic AI Dashboard**: Real-time metrics and automation performance monitoring
